# 你说我猜马上发财

一款基于 uni-app（Vue 3）的你说我猜/你比我猜小游戏。
**目前实现**「成语猜猜」模式：将手机举至额头，通过手机倾斜判定“对/跳过”，在限定时间内尽可能多猜对。

## 玩法
- 选择模式与时长（60/120/180 秒）
- 开始后把手机举到额头正前方
- 向下扣（判定为正确）/ 向上仰（判定为跳过）
- 时间结束统计得分

## 功能
- 倒计时与计分
- 设备运动传感器判定（倾斜）
- 震动反馈
- 横屏体验

## 运行方式
推荐使用 **HBuilderX** 打开本项目并运行：
1. 用 HBuilderX 打开项目目录
2. 运行到手机/模拟器（App 端）

如果你使用 uni-app CLI，请按你的项目模板与环境配置运行（本仓库未包含 `package.json`）。

## 目录结构
- `pages/index/index.vue` 核心游戏页面
- `static/data/idioms.js` 成语词库
- `pages.json` 页面与横屏设置
- `manifest.json` 应用配置

## 传感器判定说明
核心判定逻辑在 `pages/index/index.vue`。
本项目采用 **加速度计 (Accelerometer)** 方案，通过监听 **Z轴** (垂直屏幕方向) 的加速度变化来判断姿态：

- **判定依据**：Z 轴加速度 (单位：m/s²)
- **向下扣 (答对)**：`z < -5` 
  - *说明：屏幕朝下，Z 轴受重力反向拉扯，数值变为较大的负数 (实测约 -8)。*
- **向上仰 (跳过)**：`z > 7` 
  - *说明：屏幕朝上，Z 轴受重力正向叠加，数值显著变大。*
- **回正解锁 (归位)**：`-2 < z < 5`
  - *说明：手机直立贴在额头时，Z 轴几乎不受重力影响，数值接近 0。*
