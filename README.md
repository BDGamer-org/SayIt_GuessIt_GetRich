# 你说我猜马上发财

一款基于 uni-app（Vue 3）的聚会猜词小游戏。目前实现「成语猜猜」模式：将手机举至额头，通过手机倾斜判定“对/跳过”，在限定时间内尽可能多猜对。

## 玩法
- 选择模式与时长（60/120/180 秒）
- 开始后把手机举到额头正前方
- 向下扣（判定为正确）/ 向上仰（判定为跳过）
- 时间结束统计得分

## 功能
- 倒计时与计分
- 设备运动传感器判定（倾斜）
- 震动反馈
- 横屏体验 aa

## 运行方式
推荐使用 HBuilderX 打开本项目并运行：
1. 用 HBuilderX 打开项目目录
2. 运行到手机/模拟器（App 端）

如果你使用 uni-app CLI，请按你的项目模板与环境配置运行（本仓库未包含 `package.json`）。

## 目录结构
- `pages/index/index.vue` 核心游戏页面
- `static/data/idioms.js` 成语词库
- `pages.json` 页面与横屏设置
- `manifest.json` 应用配置

## 传感器判定说明
核心判定逻辑在 `pages/index/index.vue`：
- 以 `beta` 轴为主
- 向下扣：`beta > 150`
- 向上仰：`beta < 30`
- 回正解锁：`70 < beta < 110`

不同设备数值可能有差异，可在调试信息中观察并调整阈值。

## 已知事项
- 180 秒按钮当前调用的是 120 秒（可在 `pages/index/index.vue` 修正）。
- 需要设备运动传感器权限与震动权限。
